\begin{enumerate}

\section{Title}

\textbf{\Large Decision Tree and Random Forest Classification from Scratch: Spam Email Detection}

\section{Introduction}

This assignment implements two important ensemble machine learning algorithms \textbf{entirely from scratch}, without using sklearn's built-in tree classifiers:

\begin{enumerate}
    \item \textbf{Decision Tree Classifier} --- A hierarchical model that recursively splits features using information gain
    \item \textbf{Random Forest Classifier} --- An ensemble of multiple decision trees trained on bootstrap samples
\end{enumerate}

The goal is to understand the fundamental algorithms behind tree-based learning and demonstrate their application to spam email detection. Both models are implemented from first principles, including:
\begin{itemize}
    \item Entropy calculation
    \item Information gain computation
    \item Recursive tree building
    \item Bootstrap sampling
    \item Ensemble voting
\end{itemize}

\subsection{Dataset Analysis: Why Results Are as Shown}

\textbf{Data Quality Issues:}
\begin{itemize}
    \item Raw: 10 samples, both columns non-null
    \item Spam column: 3 non-null values, 7 NaN (70\% missing)
    \item Text column: 10 non-null (complete)
    \item After filtering: 3 valid samples (2 ham, 1 spam)
\end{itemize}

\textbf{Train-Test Split:}
\begin{itemize}
    \item Training: 2 samples (80\% of valid 3 samples)- 1 Ham, 1 Spam
    \item Testing: 1 sample (20\% of valid 3 samples) - 1 Spam
    \item Imbalance: Test set is 100\% minority class
    \item Challenge: Predicting from 2-sample training set is unrealistic
\end{itemize}

\subsection{What This Demonstrates}

\textbf{Root Causes:}
\begin{enumerate}
    \item Severe data quality: 70\% NaN values in spam column reduces dataset viability
    \item Critical scarcity: Only 3 valid samples after NaN removal
    \item Minimal training set: Only 2 samples for training (1 ham, 1 spam)
    \item Single-sample test: Only 1 test sample (1 spam) - insufficient for evaluation
    \item Minority class in test: Test set is 100\% spam (single sample)
    \item No ham examples in test: Both models lack data to predict ham
\end{enumerate}

\subsection{What This Demonstrates}

\textbf{Successfully Shows:}
\begin{enumerate}
    \item Algorithm mastery: From-scratch implementation proves deep understanding
    \item Data is critical: Even correct algorithms fail with inadequate data
    \item Evaluation rigor: Proper metrics reveal true model capabilities
    \item NaN handling: Demonstrates importance of data preprocessing
    \item Edge cases: Handling single-sample test sets gracefully
    \item Real-world challenges: ML in practice requires sufficient quality data
\end{enumerate}
